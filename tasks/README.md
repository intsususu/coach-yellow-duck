## 约定
- 所有任务的数据契约（颜色、文案、mock 数值、数据模型）以 PRD 为准，**不要改动数值**。
- 阶段一（T01–T08）全程用 `MockHealthRepository`，无需真机 / HealthKit。
- 每个任务文件含：目标 / 前置依赖 / 范围（做与不做）/ 详细要求 / 交付文件 / 验收标准 / PRD 引用。
- 命名、目录结构沿用 T01 建立的约定，后续任务只在自己范围内新增文件。
