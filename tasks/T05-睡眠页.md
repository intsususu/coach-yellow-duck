# T05 · 睡眠页（A4）

> 依赖：T04 ｜ PRD 引用：§5.3、§8、§6.2 睡眠数据

## 目标
实现睡眠 Tab：睡眠时长/效率 Hero + 阶段分解 + 每晚柱状图 + 事件影响。

## 范围
**做：** A4 全部。复用 T04 的图表与事件展示模式。
**不做：** 事件新增（T08）。

## 详细要求
1. **顶部**：标题「睡眠」+ 右上`＋`；时间范围切换 周/月/年（默认近 14 晚/30 天）。
2. **睡眠 Hero**：平均睡眠 `7.3h`，`↑0.2h`，效率 `95%`（可显示夜醒 8.3 次）。
3. **睡眠阶段（日均）**：深睡 `0.65h`(≈41 分) / 核心 `4.67h` / REM `1.58h`(≈100 分) / 清醒 `0.36h`；用堆叠条或环形；深睡标「偏少·建议↑」，REM 标「正常区间」。
4. **每晚时长柱状图**（`Charts/SleepChart.swift`）：取仓库 14 晚序列（分钟→小时），横轴 一~日 / 日期。
5. **事件影响**：「6月7日 · 饮酒（聚餐）— 当晚深睡 ↓、清醒增多，睡眠效率降到 88%。」（读 `appState.events` 中 drink 类）。
