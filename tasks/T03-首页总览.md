> 依赖：T02 ｜ PRD 引用：§5.1、§6.3 派生指标、§4 设计规范

## 目标
实现总览页 `HomeView`，一眼看到体重、睡眠、运动概况与本周亮点/事件洞察、近期事件。数据来自 Mock 仓库。

## 范围
**做：** A1 全部卡片与布局；从仓库读取并计算派生指标；＋记事件入口（走 T02 已有桩）。
**不做：** 图表交互细节（sparkline 静态即可）；事件弹窗真实表单（T08）。

## 详细要求（自上而下）
1. **顶部栏**：左「6月18日 星期四」+「早上好，今天」问候；右上 `＋ 记事件`（pill，调用 `presentEventEditor`）。问候语可按当前时段计算。
2. **体重 Hero 卡**（背景 `weightCardBg #eef4fd`）：
   - 当前体重 `77.1 kg`（大号 900，`brandBlue`）。
   - 右上「本周 ↓0.9 / 累计 -13.9」（绿色）。
   - 周趋势 sparkline（取仓库 W 序列尾段，Swift Charts 或 Path 静态折线）。
   - 底部「距目标 73kg · 还差 4.1kg」「查看趋势 ›」。
   - 点卡片跳体重 Tab（切 `selectedTab`）。
3. **3 圆环指标卡**：睡眠 `7.3h`（紫）、运动 `68m`（绿）、卡路里 `434 千卡`（橙），复用 `RingMetric`。
4. **关联洞察卡**（虚线框）：标题「本周亮点」+ 文案「体重连续 4 周下降，运动负荷与睡眠时长同步走高 👍」；可附事件洞察「感冒发烧那周运动暂停，体重回升 0.6kg…」。
5. **近期事件卡**：标题「近期事件」+「＋ 记录」；列出 `appState.events` 前若干条（类型色点 + 标题 + 日期）。

- 当前/累计/距目标由 W 序列与 `goalWeight` 计算，不要写死（数值应与 mock 一致 = 77.1 / -13.9 / 4.1）。
