# T06 · 运动页（A5）

> 依赖：T05 ｜ PRD 引用：§5.4、§8、§6.2 运动数据

## 目标
实现运动 Tab：消耗 Hero + 统计三元组 + 每日/每月柱状（可切指标）+ 运动类型 + 事件影响。

## 范围
**做：** A5 全部。
**不做：** 事件新增（T08）。

## 详细要求
1. **顶部**：标题「运动」+ 右上`＋`；时间范围切换；**指标切换**（千卡 / 心率），近 6 个月。
2. **运动 Hero**：日均消耗 `434 千卡`，本周 5 次 / 日均 68 分。
3. **统计三元组**：日均时长 `68 分` / 有氧心率 `121 bpm` / 周消耗 `3.0 千`（或累计 `39.5 千`千卡）。
4. **柱状图**（`Charts/ExerciseChart.swift`）：6 个月，可切 `kcal`/`hr` 指标，数据取仓库 EXM（labels/kcal/hr）。
5. **运动类型**：跑步 3 · 力量 1 · 骑行 1（或有氧占 66%，累计燃脂约 5.1kg）。
6. **事件影响**：「5月20–27日 · 腰肌肉拉伤 — 期间运动暂停，周消耗降到平时 1/3。」（读 `appState.events` injury 类）。
